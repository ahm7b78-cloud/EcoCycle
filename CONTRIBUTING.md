# Contributing to EcoCycle

We welcome contributions to EcoCycle! Here are some guidelines to help you get started:

## How to Contribute
1. **Fork the repository**: Create a personal copy of the repository on your GitHub account by forking the repo.
2. **Clone your fork**: Download a local copy of the forked repository to your computer.
3. **Create a new branch**: Always create a new branch for your work (e.g., `feature/my-feature`).
4. **Make your changes**: Implement your changes and add relevant tests if applicable.
5. **Commit your changes**: Write clear and concise commit messages that describe your changes.
6. **Push to GitHub**: Push your changes back to your GitHub fork.
7. **Open a Pull Request**: Navigate to the original repository and open a new pull request, describing your changes and the rationale behind them.

## Code Style
- Follow the code style and formatting rules in this repository.

## Testing
- Make sure to run tests to ensure everything is working as expected before submitting your pull request.

## Issues
- If you find a bug or have a feature request, please open an issue in the repository to discuss it.

Thank you for your contributions!
